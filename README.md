# Matrix Search Interface 🕶️

Este projeto é uma interface conceitual inspirada na estética do filme *The Matrix*, desenvolvida para praticar manipulação do DOM, animações com HTML5 Canvas e lógica de busca em tempo real.

## 🚀 Tecnologias Utilizadas
- **JavaScript (ES6+)**: Lógica do efeito de "chuva" e manipulação de eventos.
- **HTML5 Canvas**: Renderização de gráficos e animações fluidas.
- **Node.js**: Estrutura base para integração de serviços de backend.
- **CSS3**: Estilização temática e responsividade.

## 🧠 Habilidades Demonstradas
- **Manipulação de Canvas**: Uso de `setInterval` e coordenadas dinâmicas para criar o efeito visual clássico da Matrix.
- **Interatividade Real-time**: Captura de dados via teclado (`keyup`) com tratamento de strings (`toLowerCase`).
- **Versionamento com Git**: Gestão de repositório e fluxo de trabalho profissional.

## 🛠️ Roadmap de Evolução
- [ ] Implementar uma **API em Node.js** para transformar a busca simulada em uma busca real.
- [ ] Refatorar o código para melhorar a semântica das variáveis.
